# ETL em Python

Projeto simples feito durante um curso para exemplificar como se utilizar Python para fazer ETL.
Curso da Plataforma DIO.

Desenvolvido no Google Colab.

Parte de Extract feita a partir de um arquivo CSV com id e nome, depois foi gerada uma mensagem para cada usuário utilizando a API do gemini e depois transformado em um arquivo json com todas as informações.
